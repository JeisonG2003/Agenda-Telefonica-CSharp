# **Proyecto Agenda Telefónica C#**

Repositorio del proyecto desarrollado para la asignatura **Estructura de Datos (E)** en la **Universidad Estatal Amazónica**, carrera de **Ingeniería en Tecnologías de la Información**.

## **Descripción**

Este proyecto implementa una aplicación de consola en **C#** que simula una **Agenda Telefónica**, permitiendo registrar, almacenar, visualizar y buscar contactos de manera sencilla. Cada contacto contiene información básica como nombre, número de teléfono y correo electrónico.

El desarrollo aplica conceptos de **Programación Orientada a Objetos (POO)** y el uso de **estructuras de datos**, específicamente vectores y estructuras (`struct`), abordados hasta la semana 4 de la asignatura.  

La aplicación está organizada de forma clara, utilizando una estructura para representar los contactos y una clase principal `Program`, que gestiona el menú y la interacción con el usuario desde la consola.

Con este proyecto se demuestra el uso práctico de la POO en C#, así como la correcta organización de un proyecto ejecutable en **Visual Studio** o **Visual Studio Code**.

## **Objetivos**

* Aplicar los conceptos de **Programación Orientada a Objetos** en el desarrollo de una aplicación en C#.  
* Utilizar **estructuras de datos** como vectores y estructuras para el almacenamiento de información.  
* Desarrollar un sistema básico de Agenda Telefónica que permita agregar, mostrar y buscar contactos.  
* Demostrar la correcta estructura y funcionamiento de un proyecto de consola en C# utilizando buenas prácticas de programación.
